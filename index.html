Radio STA

Radio STA (Syiar Tauhid Aceh) is an application that provides Radio Streaming Syiar Tauhid Aceh 96.1 FM which is based on the Qur'an and Sunnah. This application also provides information in the form of radio profiles, radio streaming, radio show schedules and official radio account lists. This application was created using the React Native framework.

SIMASA

SIMASA (Sistem Informasi Masjid) is an android-based application that presents information on mosque normative data such as the year it was established, the type of mosque, address of the mosque, the name of the imam, as well as the mosque administrator. This application is created dynamically where the form and its contents can be adjusted to what is filled in the database (files, text, photos or other). This application was created using the React Native framework.

A-Tiket+

A-Tiket + is an application that provides information on events and participants of the tryout being carried out. The application also provides a ticket barcode scanning feature to facilitate attendance and a recapitulation of participants who can attend the tryout. This application also provides a feature to show all events and all participants which can be seen on the 'All Events' menu. This application is made with the Flutter framework.

KeuMaMah

Keu Mandum Ceuramah is an Android-based application with Android Native (Java) language where this application aims to make it easier for the public to see active da'i lists and can give lectures on the dates listed. This application also allows for the absence of the same schedule on each cleric and the community can know the title, place, time and cleric what gives lectures.
